Privacy Policy

Effective Date: June 3, 2026

This privacy policy details the data processing, instrumentation frameworks, and tracking architectures utilized inside the Jester Jewels mobile application ecosystem.

Installation & Campaign Attribution: We utilize the AppsFlyer SDK to measure marketing performance, attribute user installations to acquisition campaigns, and process aggregate application usage analytics to evaluate product stability.

App Tracking Transparency (ATT): On compatible iOS device environments, Jester Jewels triggers the native Apple App Tracking Transparency request layer. This operational permission enables technical token matching required exclusively for correct attribution pipelines managed via AppsFlyer.

Local State Management: All user application progress, unbolted high-scores, local game state parameters, and performance configuration settings are retained exclusively on the local hardware filesystem sandbox and are not transferred to remote clouds.

HTTPS WebView Operations: The application settings dashboard contains operational nodes that may instantiate secure, sandboxed connections via standard encrypted **HTTPS WebView** protocols to verify terms, references, or dynamic configurations on secure web hosts.
