Privacy Policy

Effective Date: June 2, 2026

This document establishes the structural disclosure paradigms regarding processing and attribution protocols integrated within Jester Jewels. We maintain structured workflows to handle device metadata strictly under verified policies.

Installation & Attribution Analytics: Jester Jewels integrates third-party software development kits, specifically AppsFlyer, to monitor, measure, and analyze marketing campaign efficiency, organic installation distributions, and aggregate operational metrics.

App Tracking Transparency (ATT): On compatible iOS deployment environments, the application requests operational clearance via the standard Apple ATT prompt. This permission authorizes token evaluations required exclusively to run structural analytics and campaign attribution workflows.

Sandbox Save Storage: All user game profiles, active milestones, high-scores, and progression tables are stored locally inside the device file sandbox structure. No remote databases maintain personal records of your progress.

Secure Remote Interfaces: The application settings dashboard contains optional triggers that may programmatically initiate secure, containerized connections via protected **HTTPS WebView** protocols to reference documentation servers.
